不用管，发错了（往上看）
　　（往前翻!第四卷没有写完!这章是发错卷了！但是番茄删不了!）孤独这两个字拆开来看，有孩童，有瓜果，有小犬，有蚊虫，足以撑起一个盛夏傍晚的巷子口，当这一切都与你无关的时候，这就叫孤独
　　——钱钟书《围城》
　　…
　　如果你因为别人的几句话就离开了你坚守好几年的事物，那你的坚守也太廉价了吧。
　　…
　　漫长的时间中，总有人将你看作宝藏。
　　…
　　莫回头看了,我们是要迈着死路去的人,没后路可看的。
　　…
　　从那天开始，我们学会明白，有些告别，就是最后一面。
　　…
　　不乱于心，不困于情，不畏将来，不念过往。如此，安好
　　…
　　幸好思念无声，否则震耳欲聋
　　…
　　眼泪这东西啊，是流出来就能把辛酸和悲伤都冲走的好东西。可等你们长大成人了就会明白，人生还有眼泪也冲刷不干净的巨大悲伤，还有难忘的痛苦让你们即使想哭也不能流泪，所以真正坚强的人，都是越想哭反而笑得越大声，怀揣着痛苦和悲伤，即使如此也要带上它们笑着前行。——《银魂》
　　…
　　时间不会帮我们解决问题，它只是把我们曾经怎么也想不通的事情，变得不再重要。
　　…
　　不要让在意你的人难过一整晚，如果你有一点点在意的话，就应该知道温度和爱意在夜晚是最容易降温的。
　　…
　　有时候，你问的问题对方一直在闪躲，那就是在变相的告诉你，真相很残忍。
　　…
　　一个人在变得铁石心肠之前，也曾付出过自己全部的温柔和善意。
　　…
　　岁月磨我少年志，时光凉我善良心，总有人间一两风，填我十万八千梦。
　　…
　　别难为自己，活的像自己就可以了。
　　…
　　每个人都在指责我，好像是我有错。他们说我需要改变。
　　但为什么是我要改变，我甚至从来没有找到过真正的自己。
　　他们却要我改变？多可笑。
　　…
　　“如何创造一个英雄呢？”“给他悲惨的童年，挫折的人生和一点点的爱”“那如何创造一个恶魔呢？”“夺去那一点点的爱”
　　…
　　开心就笑，难过就哭，烦了就骂，爱了就追，这些我们本该能轻松拥有的，却随着年纪的增长，很多就变成了奢侈。
　　…
　　悲剧并非终结，而是希望的起始
　　…
　　汝之身躯寄吾麾下，吾之命运依汝剑上
　　…
　　相遇不一定会有结局，
　　但一定会有意义。
　　在追逐月光的过程中，
　　别忘了，你也曾被月光照耀过。
　　…
　　我曾七次鄙视过自己的灵魂
　　第一次，当它本可努力时，却故作谦虚
　　第二次，它怨天尤人，无病呻吟
　　第三次，在困难和容易之间，它选择了容易
　　第四次，它犯了错却借由别人也会犯错来安慰自己
　　第五次，它因为懦弱而忍让，却声称为自己的忍让
　　第六次，当它鄙视一张丑恶的嘴脸时，却不知那正是自己面具的其中一副
　　第七次，它侧身于生活的污泥中，虽不甘心，却又畏首畏尾。
