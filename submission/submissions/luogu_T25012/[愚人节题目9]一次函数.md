# [愚人节题目9]一次函数

## 题目描述

后台有一个一次函数$y=kx+b$，你的程序每次能读到一个$x$，你应输出相应的$y$。一次函数中$k$和$b$是常数，所有数字均为整数。

$|k|,|b| \leq 100$

$0 \leq x \leq 100$

如果你的程序与标准输出的绝对值差为$dis$，那么你会获得$\frac {max(10000 -
 dis^{2},0)}{10000} \times 100\%$的分数。

（但是在ACM赛制中，没有全部AC即为0分）

## 输入格式

输入一个数字$x$

## 输出格式

输出一个数字$y$
