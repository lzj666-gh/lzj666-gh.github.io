# 素数个数

## 题目描述

小明会告诉你一个数$n\leq3×10^8$,你需要告诉他小于该数的素数个数。请注意程序的时间常数复杂度与~~使用空间换时间时所用的空间的~~空间复杂度。

## 输入格式

一行一个整数$n$。

## 输出格式

一行一个整数，要求如题目所示。

## 提示

| 测试点 | 满足条件 | 限时 |
| -----------: | -----------: | -----------: |
| 1-4 | $n\leq$1E+06 | 0.5s |
| 5-7 | $n\leq$3E+08 | 2.0s |

因洛谷测评机速度波动，请在#5,#6,#7死掉的稍微等待一下后重新提交。如时间复杂度明显不合格，请勿卡评测。

## 时空限制

时间限制: 50 ms
内存限制: 38 MB
