# [SDOI2012] Longge 的问题

## 题目描述

现在问题来了：给定一个整数 $n$，你需要求出 $\sum\limits_{i=1}^n \gcd(i, n)$，其中 $\gcd(i, n)$ 表示 $i$ 和 $n$ 的最大公因数。  

## 输入格式

输入只有一行一个整数，表示 $n$。

## 输出格式

输出一行一个整数表示答案。

## 提示

#### 数据规模与约定

- 对于 $60\% $ 的数据，保证 $n\leq 2^{16}$。
- 对于 $100\% $ 的数据，保证 $1\leq n< 2^{32}$。

## 时空限制

时间限制: 3000 ms
内存限制: 125 MB
