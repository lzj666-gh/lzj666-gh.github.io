长长的二维码

题目描述

他要造的长条形二维码实在太长了，而且有很多行是重复的！为了保证他的电脑不炸掉，他将其转换成了一些代码。1表示黑色单位（“#”），0表示白色单位（" "），每个串会有一个代号，为一个数字。他会先告诉你有多少个代号和有多少行。问这个二维码是什么？

例如：
1 10101010101010111
2 10101010101010101
则：
1表示：# # # # # # # ###
2表示：# # # # # # # # #
样例中：
2 3
1 10101010101010111
2 10101010101010101
1
2
1
表示：
# # # # # # # ###
# # # # # # # # #
# # # # # # # ###

输入格式

第一行两个数nnn,mmm，表示有nnn个代号，mmm行

接下来n行，每行一个数AiA_iAi​和一个零一串SiS_iSi​，表示代号为AiA_iAi​的字符串转化后为SiS_iSi​。

接下来mmm行，每行一个代号，用以代表该行的字符串。

输出格式

mmm行，内容如题目描述所示。

提示

mmm<=1000,nnn<=100000

AiA_iAi​<=1E+12,SiS_iSi​长度<=1E+05

时空限制

时间限制: 8 ms
内存限制: 128 MB
