# path

## 题目描述

给定长度为n的数列a，如果ai & aj $\neq$ 0（按位与），则在i,j之间存在一条长度为ai+aj的边，求1至所有点的最短路。

## 输入格式

第一行一个正整数n。
接下来一行n个整数a1~an。


## 输出格式

输出一行n个整数，第i个为1到i的最短路长度，不能到达输出 -1。

## 提示

对于 20% 的数据，n≤10。\
对于 40% 的数据，n≤10^3。\
另外 20% 的数据，所有的ai均可表示为$2^x+2^y$的形式。\
对于 100%的数据，1≤n≤10^5，0≤ai≤2^30。


## 时空限制

时间限制: 1000 ms
内存限制: 128 MB
