# 0x0练习10-进制转换-数的进制转换

## 题目描述

一个数N从A进制转换到B进制。

## 输入格式

第一行是一个数T，代表测试数据组数。
接下来的T行中有一个数A，一个数B，以及A进制下的数N。
我们对N的表示方法：扩展数字表，现在的数字由小到大依次是0-9,A-Z,a-z共62个。
也就是说：1 = 1, ..., 9 = 9, A = 10, B = 11, ..., Z = 35, a = 36, b = 37, ..., z = 61。

## 输出格式

每组数据三行：
第一行是A，以及A进制下的数字N。
第二行是B，以及B进制下的数字N。
第三行是一个空行。

## 时空限制

时间限制: 1000 ms
内存限制: 64 MB
