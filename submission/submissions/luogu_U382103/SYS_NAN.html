SYS_NAN

题目描述

所以他来求助你一道数学题：

这道题会给你 nnn 个操作，有一个初始值 kkk。

接下来 nnn 行，会有一个操作符（假设为 ccc）与一个数（假设为 ppp）：

如果操作符为 那么
+ k=k+p\sqrt{k+p}k+p​
- k=k−p\sqrt{k-p}k−p​
* k=kp\sqrt{\frac{k}{p}}pk​​
/ k=k×p\sqrt{k \times p}k×p​
% k=k mod p\sqrt{k \bmod p}kmodp​

如果无法计算出答案（例如根号下的数为负数、除 0……），输出 Nan 并退出。

输入格式

第一行：输入 n kn \space kn k 如题目描述。

接下来 nnn 行每行一个操作符与一个数，如题目描述。

输出格式

nnn 行，每行输出这次操作后的 kkk。
