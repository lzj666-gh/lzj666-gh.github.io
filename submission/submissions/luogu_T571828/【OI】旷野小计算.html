【OI】旷野小计算

题目描述

关于什么是寄存器：理解成变量即可。

您需要编写指令，完成给定的计算任务。

在以下指令中，小写字母 v 开头的参数表示十进制无符号常数，大写字母 R 开头的参数表示寄存器。寄存器均为 64 位无符号整数。您可以自由使用的寄存器有六个：a、b、c、d、x、y。除非特别说明，否则所有寄存器初始值均为 000。

指令格式 描述
+ R1 R2 使 R1 寄存器加上 R2 的值
- R1 R2 使 R1 寄存器减去 R2 的值
< R1 v 使 R1 寄存器左移 vvv 位，低位补 0
> R1 v 使 R1 寄存器右移 vvv 位，高位补 0
^ R1 R2 使 R1 寄存器异或上 R2 的值
! R1 输出 R1 寄存器的值
= R1 v 使 R1 寄存器的值变为 vvv
* R1 R2 使 R1 寄存器乘上 R2 的值
/ R1 R2 使 R1 寄存器除以 R2 的值

提示

请注意：

  • 程序的大小非常关键，运行的指令条数超过了 PC 数限制就会扣除分数。PC 数限制 x+yx\color{green}+yx+y 表示：

    • 当运行的指令条数 c≤xc \le xc≤x 时，获得该测试点的全部分数；
    • 当运行的指令条数 c>x+yc >x+yc>x+y 时，不得分；
    • 当运行的指令条数 c∈(x,x+y)c \in (x, x+y)c∈(x,x+y) 时，获得该测试点 (x+y−c)/y×100%(x+y-c)/y\times 100\%(x+y−c)/y×100% 的分数（即：线性衰减）。

  • 常数、乘法、除法指令（即表格中后三条指令）是非常昂贵的，使用了就会扣除所有的分数。

因此，两个样例的输出实际均不能得到分数。
