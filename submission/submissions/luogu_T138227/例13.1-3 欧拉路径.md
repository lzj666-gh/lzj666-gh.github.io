# 例13.1-3 欧拉路径

## 题目描述

给定一个无向图，请你判断它是欧拉图、半欧拉图还是非欧拉图。

## 输入格式

第一行包含两个整数 N 和 M，表示无向图的点和边的数量。 

接下来 M 行，每行包含两个整 数 a,b，表示点 a 和 b 之间存在一条边。  

所有点的编号从 1∼N。


## 输出格式

首先，在第一行按顺序输出点$1∼N$中每个点的度数。  

第二行输出对该图的判断，Eulerian（欧拉图），Semi-Eulerian（半欧拉图），Non-Eulerian（非欧拉图）。


## 提示

$1≤N≤500，1≤M≤N(N−1)/2  $


## 时空限制

时间限制: 1000 ms
内存限制: 128 MB
