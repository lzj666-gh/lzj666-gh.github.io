[HNOI2006] 公路修建问题

题目描述

OI island 是一个非常漂亮的岛屿，自开发以来，到这儿来旅游的人很多。然而，由于该岛屿刚刚开发不久，所以那里的交通情况还是很糟糕。所以，OIER Association组织成立了，旨在建立 OI island 的交通系统。

OI island 有 nnn 个旅游景点，不妨将它们从 111 到 nnn 标号。现在，OIER Association 需要修公路将这些景点连接起来。一条公路连接两个景点。公路有两种类型，不妨称它们为一级公路和二级公路。一级公路上的车速快，但是修路的花费要大一些。

OIER Association 打算修 n−1n-1n−1 条公路将这些景点连接起来（使得任意两个景点之间都会有一条路径）。为了保证公路系统的效率, OIER Association 希望在这 n−1n-1n−1 条公路之中,至少有 k(0≤k≤n−1)k(0\le k\le n-1)k(0≤k≤n−1) 条一级公路。OIER Association 也不希望为一条公路花费太多的钱。所以，他们希望在满足上述条件的情况下，花费最多的一条公路的花费尽可能的少。而你的任务就是，在给定一些可能修建的公路的情况下，选择 n−1n-1n−1 条公路，满足上面的条件。

输入格式

文件第一行有三个数 n(1≤n≤10000)n(1 \le n \le 10000)n(1≤n≤10000)，k(0≤k≤n−1),m(n−1≤m≤20000)k(0 \le k \le n-1),m(n-1 \le m \le 20000)k(0≤k≤n−1),m(n−1≤m≤20000)，这些数之间用空格分开。NNN 和 kkk 如前所述，mmm 表示有 mmm 对景点之间可以修公路。

以下的 m−1m-1m−1 行，每一行有 444 个正整数 a,b,c1,c2a,b,c_1,c_2a,b,c1​,c2​，（1≤a,b≤n,a≠b,1≤c2≤c1≤300001 \le a,b \le n,a \ne b,1 \le c_2 \le c_1 \le 300001≤a,b≤n,a=b,1≤c2​≤c1​≤30000），表示在景点 aaa 与 bbb 之间可以修公路，如果修一级公路，则需要 c1c_1c1​ 的花费，如果修二级公路，则需要 c2c_2c2​ 的花费。

在实际评测时，将只会有 m−1m-1m−1 行公路

输出格式

输出第一行有一个数据，表示花费最大的公路的花费。

接下来的 n−1n-1n−1 行，每行有两个正整数 ttt 和 ppp，ttt 和 ppp 表示在输入的第 ttt 对（按照输入的顺序，从 111 开始标号）景点之间修建 ppp 级公路。

n−1n-1n−1 条公路的输出必须按 ttt 的大小递增，如果有多个方案使花费最大的公路花费最小，那么输出任意一个都可以。

时空限制

时间限制: 1000 ms
内存限制: 125 MB
