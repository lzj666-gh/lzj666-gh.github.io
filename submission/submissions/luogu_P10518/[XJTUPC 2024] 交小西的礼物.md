# [XJTUPC 2024] 交小西的礼物

## 题目描述

小西准备给 $a$ 名志愿者，$b$ 支单挑队，$c$ 支双人队，$d$ 支三人队的**每个人**恰好发一份奖品，她想知道一共要发几份奖品？


## 输入格式

输入数据共一行四个整数 $a,b,c,d$ ($0 \leq a, b, c, d \leq 200$)，含义如上所述。

## 输出格式

输出数据共一行一个整数，表示总共要发的奖品数。
