### 那么就规规矩矩地用for循环吧
## 话不多说，上代码
```
#include<iostream>
using namespace std;
int main()
{
    int a=0,n;//定义，a初始化为0
    cin>>n;//输入
    for(int i=1;i<=n;i++)//循环，从一到n
    {
    	a+=i;//累加
    	}
    	cout<<a;//输出
    	return 0;//好习惯
    	}	
```
