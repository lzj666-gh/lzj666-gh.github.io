String Minimization

题目描述

你有四个长 nnn 的字符串 a,b,c,da,b,c,da,b,c,d。你可以执行任意多次如下操作：

  • 选择一个 iii，交换 ai,cia_i,c_iai​,ci​，然后交换 bi,dib_i,d_ibi​,di​。

求在 aaa 的字典序尽量小的前提下，bbb 字典序最小是什么。

如果你不知道什么是字典序，看这里：

对于两个字符串 p,qp,qp,q，称 ppp 的字典序小于 qqq（记为 p<qp<qp<q），当且仅当存在自然数 kkk 使 p,qp,qp,q 的前 kkk 个字符相同且 pk+1p_{k+1}pk+1​ 的 ASCII 码小于 qk+1q_{k+1}qk+1​ 的 ASCII 码。

例如：

  • abc<baa\texttt{abc}<\texttt{baa}abc<baa（当 k=0k=0k=0）
  • bae<bbb\texttt{bae}<\texttt{bbb}bae<bbb（当 k=1k=1k=1）

输入格式

输入的第一行有一个正整数 nnn，表示字符串 a,b,c,da,b,c,da,b,c,d 长度。

之后四行，每行一个字符串，分别表示 a,b,c,da,b,c,da,b,c,d。

输出格式

输出一行一个字符串，表示题目要求的字符串 bbb。

提示

【样例解释】

选择 iii 为 1,3,41,3,41,3,4 可以让 aaa 取到最小的字典序 weablake\texttt{weablake}weablake，此时字符串 bbb 也得到满足题意最小的字典序 auazyqaq\texttt{auazyqaq}auazyqaq。

事实上如果 i=1i=1i=1 时不操作 aaa 的字典序也是最小的，但是此时字符串 bbb 就是 yuazyqaq\texttt{yuazyqaq}yuazyqaq，不够小。
