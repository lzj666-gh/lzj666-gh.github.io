【模板】二维线段树

题目描述

给定一个大小为 n×mn\times mn×m 的矩阵。

会不停给出下列两种询问，直至文件结束。

1 a b c d x，表示将左上角为(a,b)(a,b)(a,b)，右下角为 (c,d)(c,d)(c,d) 的子矩阵全部加上 xxx；

2 a b c d，表示询问左上角为 (a,b)(a,b)(a,b)，右下角为 (c,d)(c,d)(c,d) 为顶点的子矩阵的所有数字之和。

输入格式

第一行两个正整数，其中 n,mn,mn,m 分别表示矩阵的行数与列数。

接下来若干行直到文件结束，代表你需要进行的操作。

输出格式

对于每个 222 操作，输出一行代表查询的结果。

提示

对于 10%10\%10% 的数据，1≤n,m≤161 \le n,m \le 161≤n,m≤16，操作不超过 200200200 个；

对于 100%100\%100% 的数据，1≤n,m≤2048,∣x∣<=5001\le n,m\le 2048,|x|<=5001≤n,m≤2048,∣x∣<=500，操作不超过 2×1052\times 10^52×105 个，保证运算过程中及最终结果均不超过 long long 的表示范围，并且修改与查询的子矩阵存在。

时空限制

时间限制: 2000 ms
内存限制: 512 MB
