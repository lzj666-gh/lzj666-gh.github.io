# B2025 题解

## 题意分析
本题主要考察了我们对各种语言中输出函数的理解，在这里提供3种语言的代码供参考。

## 代码
1. PHP

	PHP 有一个特性，就是 PHP 标签外字符会自动输出，所以我们就只要把题目里的输出黏贴就可以啦！

	代码：
	```php
	  *
	 ***
	*****
	 ***
	  *
	```

2. Python

	在 Python 中，我们使用 print 函数进行输出。

	代码:
	```python
	print("  *")
	print(" ***")
	print("*****")
	print(" ***")
	print("  *")
	```

3. 文言
	
    ~~为什么看起来像打网游的时候骂了脏话然后被屏蔽了。~~

	使用吾有一言。曰「「  *」」。書之。即可。

	 代码:
	```
	吾有一言。曰「「  *」」。書之。
	吾有一言。曰「「 ***」」。書之。
	吾有一言。曰「「*****」」。書之。
	吾有一言。曰「「 ***」」。書之。
	```
