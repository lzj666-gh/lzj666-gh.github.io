Vector_Math选路

题目描述

他建立了 nnn 个标记点.

1号标记点是 Vector_Math 的家，nnn 号标记点是目的地.

Vector_Math 已经知道了
一些标记之间的距离，以正常人的思路来看，肯定是走路程最短的，可 Vector_Math 不这样看.

他会从路程前 kkk 短里随机选一条路.

输入格式

第一行三个数，nnn,mmm 与 kkk，表示共有 nnn 个标记点与 mmm 条路，要求前 kkk 短的路程.

接下来 mmm 行，每行三个数，ui,vi,wiu_i,v_i,w_iui​,vi​,wi​.

表示从 uiu_iui​ 标记点到 viv_ivi​ 标记点有一条权值为 wiw_iwi​ 的单行道.

输出格式

输出一行，kkk 个数，表示 1−>n1->n1−>n 前 kkk 短路.

如果不能构成 kkk 短路，能输出就输出.

提示

1≤n≤5000，1≤m≤100000，1≤k≤100001 \le n \le 5000，1 \le m \le 100000，1 \le k \le 100001≤n≤5000，1≤m≤100000，1≤k≤10000

前 kkk 短路可以走环.

注意此题道路均为单行道.

样例解释：

第一短: 1−>41->41−>4 路程为 111.

第二短: 1−>2−>1−>41->2->1->41−>2−>1−>4 路程为 333.

第三短: 1−>4−>1−>41->4->1->41−>4−>1−>4 路程为 333.

数据限制只有 600ms(0.6s)600ms(0.6s)600ms(0.6s)，用普通搜索剪枝不能过哟.

时空限制

时间限制: 600 ms
内存限制: 128 MB
