【OI】连号子序列数

题目描述

小 S 这些天一直在思考这样一个奇怪而有趣的问题：

在 1∼N1\sim N1∼N 的某个全排列中有多少个连号子序列呢？这里所说的连号子序列的定义是：

如果一个子序列里的所有元素递增排序后能得到一个“连续”数列，则称这个子序列连号子序列。

当 NNN 很小的时候，小 S 可以很快地算出答案，但是当 NNN 变大的时候，问题就不是那么简单了，现在小 S 需要你的帮助。

输入格式

第一行是一个正整数 N(1≤N≤500000)N (1 \le N \le 500000)N(1≤N≤500000)，表示全排列的规模。

第二行是 NNN 个不同的数字 Pi(1≤Pi≤N)P_i(1 \le P_i \le N)Pi​(1≤Pi​≤N)，表示这 NNN 个数字的某一全排列。

输出格式

输出一个整数，表示不同连号子序列的数目。

提示

对于 20%20\%20% 的数据，N≤20N\le 20N≤20。

对于 40%40\%40% 的数据，N≤500N\le 500N≤500。

对于 60%60\%60% 的数据，N≤5000N\le 5000N≤5000。

对于 100%100\%100% 的数据，N≤500000N\le 500000N≤500000。

时空限制

时间限制: 1000 ms
内存限制: 512 MB
