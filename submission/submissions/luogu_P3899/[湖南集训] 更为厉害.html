[湖南集训] 更为厉害

题目描述

设 T\text TT 为一棵有根树，我们做如下的定义：

  • 设 aaa 和 bbb 为 T\text TT 中的两个不同节点。如果 aaa 是 bbb 的祖先，那么称“aaa 比 bbb 更为厉害”。
  • 设 aaa 和 bbb 为 T\text TT 中的两个不同节点。如果 aaa 与 bbb 在树上的距离不超过某个给定常数 xxx，那么称“ aaa 与 bbb 彼此彼此”。

给定一棵 nnn 个节点的有根树 T\text TT，节点的编号为 111 到 nnn，根节点为 111 号节点。
你需要回答 qqq 个询问，询问给定两个整数 ppp 和 kkk，问有多少个有序三元组 (a,b,c)(a,b,c)(a,b,c) 满足：

  1. a,b,ca,b,ca,b,c 为 T\text TT 中三个不同的点，且 aaa 为 ppp 号节点；
  2. aaa 和 bbb 都比 ccc 更为厉害；
  3. aaa 和 bbb 彼此彼此。这里彼此彼此中的常数为给定的 kkk。

输入格式

输入文件的第一行含有两个正整数 nnn 和 qqq，分别代表有根树的点数与询问的个数。

接下来 n−1n - 1n−1 行，每行描述一条树上的边。每行含有两个整数 uuu 和 vvv，代表在节点 uuu 和 vvv 之间有一条边。

接下来 qqq 行，每行描述一个操作。第 iii 行含有两个整数，分别表示第 iii 个询问的 ppp 和 kkk。

输出格式

输出 qqq 行，每行对应一个询问，代表询问的答案。
