# 骑士的工作

## 题目描述

每个人都可以砍掉**至多**一个大小不超过 $z_i$ 的头，需要 $z_i$ 个金币，求最小花费。


## 输入格式

第一行两个整数 $n$，$m$。

下接 $n$ 行，一个整数表示 $n$ 个头的大小。

下接 $m$ 行，每个人可以砍的头大小和需要的金币数 $z_1, \cdots, z_m$。


## 输出格式

一个整数，最小花费。如果无解，输出 `you died!`。


## 提示

对于所有数据，$1 \le n,m \le 2 \times 10^4$，$1 \leq z_i \leq 10^5$。

## 时空限制

时间限制: 1000 ms
内存限制: 125 MB
