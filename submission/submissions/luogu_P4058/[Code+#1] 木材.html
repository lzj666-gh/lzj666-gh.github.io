[Code+#1] 木材

题目描述

有 nnn 棵树，初始时每棵树的高度为 HiH_iHi​，第 iii 棵树每月都会长高 AiA_iAi​。现在有个木料长度总量为 SSS 的订单，客户要求每块木料的长度不能小于 LLL，而且木料必须是整棵树（即不能为树的一部分）。现在问你最少需要等多少个月才能满足订单。

输入格式

第一行 333 个用空格隔开的非负整数 n,S,Ln,S,Ln,S,L，表示树的数量、订单总量和单块木料长度限制。

第二行 nnn 个用空格隔开的非负整数，依次为 H1,H2,...,HnH_1,H_2, ... ,H_nH1​,H2​,...,Hn​。

第三行 nnn 个用空格隔开的非负整数，依次为 A1,A2,...,AnA_1,A_2, ... ,A_nA1​,A2​,...,An​。

输出格式

输出一行一个整数表示答案。

提示

对于样例，在六个月后，各棵树的高度分别为 14,47,5614,47,5614,47,56，此时无法完成订单。

在七个月后，各棵树的高度分别为 16,54,6516,54,6516,54,65，此时可以砍下第 222 和第 333 棵树完成订单了。
