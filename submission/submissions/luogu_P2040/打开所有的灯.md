# 打开所有的灯

## 题目描述

这个灯很奇 (fan) 怪 (ren)，点一下就会将这个灯和其周围四盏灯的开关状态全部改变。现在你的任务就是就是告诉 pmshz 要全部打开这些灯。

例如
```plain
0  1  1
1  0  0
1  0  1
```
点一下最中间的灯 $(2,2)$ 就变成了

```plain
0  0  1
0  1  1
1  1  1
```

再点一下左上角的灯 $(1,1)$ 就变成了

```plain
1  1  1
1  1  1
1  1  1
```

达成目标。

最少需要 $2$ 步。

## 输入格式

$9$ 个数字，以 $3\times3$ 的格式输入，每两个数字中间只有一个空格，表示灯初始的开关状态。（$0$ 表示关，$1$ 表示开）

## 输出格式

一个整数，表示最少打开所有灯所需要的步数。
