CSP2024 游寄

初赛

你说得对，但是前面忘了，中间忘了，后面忘了，但是随便过。

复赛

Day 0

熬夜打原神，原神真好玩。 不敢熬夜，赶紧睡觉，我睡睡睡睡睡，直接一路睡到考场（雾）。

Day 1

到了七中高新，左右看了看，没发现几个同届的，倒发现几个学弟学妹。听到有人在讨论怎么骗分，不是吧，-J就骗？随便和学弟学妹交流了一下心得，发现能进考场了，就进去了。

-J

好，今年居然能考前上机测试，直接花15min打+调一颗线段树，然后发现同机房的都在玩？例如一个同校的学弟就在玩小恐龙，别说，打的比我好（大雾）。

T1

水，比去年水， map 打完走人。

T2

比去年水， dfs 打完走人（吗？）。

不对，大样例怎么爆栈了？！吓得我当场原地旋转720°加后空翻震惊监考老师的同时开了无限栈，诶，不对，无限栈咋开来着，哦，只需要这样，这样，再那样就可以了，最后也是过了大样例，吓我一跳。

T3

先写了个记忆化递归打表程序，发现十分有规律，就是  mod 7\bmod 7mod7 然后特判就行了，做到现在过完大样例大约 0.75h 左右，开 T4！

T4

不对！这 T4 不对劲，我看着起码有上位绿，甚至有蓝，完了，是不是我思考复杂了，再推简单思路。

只剩 0.5h 了，寄，赶紧打个暴力的区间 DP，感觉要遭。

-S

接着打了棵 FHQ，还没有调呢就听见这个考场有人根据大样例推测名字为 duel 的题是输出序列中出现最多的元素出现多少次，不是，赛前 10min 写完 T1 正解，这个人怕不是开了。

T1

最后想着用 set 维护，还是思路有正确性的。

T2

感觉还是大模拟，随便看了几下，口胡了一个思路，几下过了大样例，开 T3，T4 去。

T3

先把暴力打上再说。去 T4 看一眼……WC，什么玩意，不会，再来打 T3，思考了 0.114514ms 后觉得先把 T4 暴力打上再说。然后就回不来了。

T4

咋感觉起码有紫，暴力打完了后调了很久都不过样例，一直调到结束，期间检查了 1919810 次 freopen。

Day2

出难度了，S 组是什么东西！算了，明年再战吧。

-J=100+100+100+20\texttt{-J} = 100 + 100 + 100 + 20-J=100+100+100+20

-S=100+100+20+0\texttt{-S} = 100 + 100 + 20 + 0-S=100+100+20+0

Day8

出成绩了，J 组的 T4 挂了 5pts，其他倒没挂。

赛后总结

-J T4

赛后看了题解，感觉区间 DP 做法还是赛时可做的，但是局限于我最开始的图论做法，而且不想着怎么去优化，区间 DP 也想过，但是没想过怎么压缩状态，总结下来就是思路想不到，迷途不知返，以后一定要多尝试几种思路。

-S T3

不对啊，同机房的大佬都想到做法了，我真是个蒟蒻。赛后问了一下解法，发现怎么又是区间 DP，看来我接下来这一年要好好学一下区间 DP 了，不然普及提高全部挂分，挂大分。fc 了一下我的思路要转到区间 DP 上有多难，其实差距也只有亿点大罢了，只需要转移一个最近的同色和异色，wssb。

-S T4

真的不会了，在我眼里这道题几乎全部不可做，最暴力的暴力都不会打，看的出来我是真的蒟了。

感言

本来还以为要退队了呢，还好这次挂的分少，得以幸存（？）。总结一下就是区间 DP 和 Just do it，几种思路就都写上也行。

咕咕咕。
