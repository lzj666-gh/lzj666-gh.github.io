CF1842B 题解

题目大意

给定三个栈，每个栈有 nnn 个非负数，初始值为 u=0u=0u=0，每取一个数 vvv，u←u∣vu \leftarrow u|vu←u∣v，问最终结果是否能达到 xxx。

分析

由于是按位或运算，二进制中 0 的部分在或了以后是会根据对方的值来影响结果的。

所以就有贪心策略：只要栈顶的元素在或上 xxx 以后还等于 xxx，即 xxx 二进制位上为 0 的部分其二进制位上也为 0 都选。
