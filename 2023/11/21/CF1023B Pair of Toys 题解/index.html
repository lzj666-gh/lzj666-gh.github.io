CF1023B Pair of Toys 题解

简化题意

求 1 到 nnn 中取两个数，问有几种方案。

做法

Unaccepted\color{red}{Unaccepted}Unaccepted

直接暴力，枚举两个数（因为题目中说了选 a b 和选 b a 都是相同的，所以可以直接枚举所有 a<ba < ba<b 的 a b）。

正解

刚刚上面的做法太暴力了，因为它 Θ(n2)\Theta(n^2)Θ(n2) 的时间复杂度以及 1≤n,k≤10141 \le n,k \le 10^{14}1≤n,k≤1014 的数据范围 TLE 掉了。

现在我们先进行一些数学的证明：

∵a+b=a+b\because a + b = a + b ∵a+b=a+b

∴a+b+c−c=a+b\therefore a + b + c - c = a + b ∴a+b+c−c=a+b

∴a+c+b−c=a+b\therefore a + c + b - c = a + b ∴a+c+b−c=a+b

∴(a+c)+(b−c)=a+b\therefore ( a + c ) + ( b - c ) = a + b ∴(a+c)+(b−c)=a+b

所以我们知道当 a+b=ka+b=ka+b=k 时 (a+1)+(b−1)=k( a + 1 ) + ( b - 1 ) = k(a+1)+(b−1)=k。

而且第一组物品就是 0 k - 1。

但是如果 kkk 大于 n×2−1n \times 2 - 1n×2−1 就无解了。

因为最大的一组是 n n-1，所以最大的一组和就是 n+n−1n + n - 1n+n−1 ,kkk 大于它就无解了。

于是判断一下 kkk 的奇偶性再在第一个数和第二个数的个数之间取 min⁡\minmin 就行了。
