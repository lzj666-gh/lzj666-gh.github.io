题目：彩色项链计数问题

题目描述
小美有一串由 nnn 颗彩色珠子组成的项链。每颗珠子可以染成 mmm 种颜色中的一种。项链可以旋转，但不能翻转。定义一种染色方案的美学值为所有的子串中的出现次数最多的颜色种数之和。

例如：当 n=3,m=2n=3, m=2n=3,m=2 时，染色方案 ABA 的美学值为：

  • k=1k=1k=1 时有 A, B, A（共 3 个贡献）
  • k=2k=2k=2 时有 AB, BA （共 4 个贡献）
  • k=3k=3k=3 时有 ABA（1 个同色）
    总美学值为 0+1=10 + 1 = 10+1=1。

请你计算所有可能的染色方案的美学值之和，答案对 998244353998244353998244353 取模。

输入格式
一行两个整数 n,mn, mn,m（2≤n≤105,1≤m≤1092 \leq n \leq 10^5, 1 \leq m \leq 10^92≤n≤105,1≤m≤109）

输出格式
输出一个整数表示答案

样例输入

样例输出

部分分设置

  • 50% 数据：n≤1000n \leq 1000n≤1000
  • 100% 数据：n≤105n \leq 10^5n≤105

设计思路

  1. 知识结合：本题综合了中提到的Polya定理和中的NTT技术，实现计数问题的优化
  2. 难度梯度：设置暴力解法保底分，满分需要深入理解群论与生成函数的结合应用
  3. 思维层次：
    • 第一层：理解旋转等价的计数原理
    • 第二层：将美学值转化为生成函数模型
    • 第三层：通过多项式快速幂实现高效计算
